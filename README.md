\# Career Roadmap



Structured learning roadmap, notes, labs, and portfolio development for Technical Solutions Engineer, DevOps, and backend/software roles.



\## Purpose

This repository is being used to:

\- build technical foundations step by step

\- keep learning notes in markdown

\- organise practical labs and exercises

\- track weekly progress

\- create visible portfolio proof on GitHub



\## Current structure

\- `00\_admin` - goals, role targets, weekly plan, and project control files

\- `01\_notes` - learning notes

\- `02\_labs` - hands-on practice files

\- `04\_screenshots` - proof screenshots organised by week

\- `05\_weekly\_reviews` - short weekly review files



\## Current stage

Week 1 - terminal basics, PowerShell basics, Git setup, and first GitHub push
