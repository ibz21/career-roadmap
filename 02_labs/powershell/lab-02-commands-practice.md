\# Lab 02 - Commands Practice



\## Goal

Practice the first basic PowerShell commands and understand what they do.



\## Commands practised



\### cd

Change directory.

Used to move into another folder.



\### dir

Directory listing.

Used to show what is inside a folder.



\### mkdir

Make directory.

Used to create a folder.



\### New-Item

Used to create a new file or item.



\### pwd

Print working directory.

Used to show my current location.



\## What I learned

\- A folder is also called a directory

\- A file and a folder are different things

\- The terminal always has a current location

\- I can work in another folder directly by using a path



\## Why this matters

These commands are the first layer of terminal confidence.

They help me move around, inspect project structure, and create files and folders without using the mouse.
