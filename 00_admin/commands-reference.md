\# Commands Reference



\## cd

Change directory.

Used to move into another folder.



\## dir

Directory listing.

Used to show files and folders in a location.



\## mkdir

Make directory.

Used to create a folder.



\## New-Item

Create a new item.

Used to create a file or folder.



\## pwd

Print working directory.

Used to show my current location.



\## Key ideas

\- A directory is a folder

\- A path is the address of a file or folder

\- . means the current location

\- .. means the folder above the current location
