\# Project Structure



\## Main folders



\### 00\_admin

Project control files such as goals, role targets, weekly plan, and command references.



\### 01\_notes

Learning notes written in markdown.



\### 02\_labs

Hands-on practice files and small exercises.



\### 04\_screenshots

Proof screenshots organized by week.



\### 05\_weekly\_reviews

Short review files showing what I completed and what still feels new.
